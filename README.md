# GL Transport Express - Site Oficial

## Descrição

Site profissional para a GL Transport Express, empresa especializada em entregas rápidas e coletas em toda a Grande São Paulo. O site apresenta serviços, permite contato direto via WhatsApp e oferece uma experiência moderna e responsiva.

## Funcionalidades

### Características Principais
- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Header Sólido**: Design profissional sem transparência
- **Navegação Intuitiva**: Menu fácil de usar com badges de destaque
- **Integração WhatsApp**: Contato direto com mensagem personalizada
- **Formulário de Contato**: Captura leads e redireciona para WhatsApp
- **Animações Modernas**: Efeitos suaves e transições elegantes
- **SEO Otimizado**: Meta tags e estrutura semântica

### Seções do Site
1. **Header**: Contato WhatsApp e navegação principal
2. **Hero**: Apresentação principal com call-to-action
3. **Serviços**: Cards detalhados dos serviços oferecidos
4. **Serviços Adicionais**: Serviços complementares com ícones animados
5. **Avaliações**: Depoimentos de clientes
6. **Estatísticas**: Números da empresa
7. **Quem Somos**: Informações sobre a empresa
8. **Contato**: Formulário e informações de contato
9. **Footer**: Links úteis e mapa interativo

## Tecnologias Utilizadas

### Frontend
- **HTML5**: Estrutura semântica moderna
- **CSS3**: Estilos com animações e design responsivo
- **JavaScript ES6+**: Funcionalidades interativas
- **Font Awesome 6.0**: Ícones profissionais

### Design
- **Cores Principais**:
  - Azul escuro: `#2c3e50` (header)
  - Verde WhatsApp: `#25d366` (botões e links)
  - Branco: `#ffffff` (textos e fundos)
- **Tipografia**: Arial, sans-serif
- **Layout**: Grid e Flexbox

## Personalização

### Cores
Para alterar as cores principais, modifique no `style.css`:

```css
:root {
    --primary-color: #2c3e50;    /* Azul escuro */
    --whatsapp-color: #25d366;   /* Verde WhatsApp */
    --text-color: #ffffff;       /* Branco */
    --secondary-color: #3498db;  /* Azul claro */
}
```

### WhatsApp
Para alterar o número de WhatsApp, atualize em:
1. `index.html`: Todos os links `wa.me/`
2. `script.js`: URLs do WhatsApp

### Logo
Substitua `assets/img/logo-Photoroom.png` pela nova logo.

## Instalação e Uso

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web (Apache, Nginx, ou similar)
- Conexão com internet (para Font Awesome)

### Instalação Local
1. Clone ou baixe os arquivos
2. Abra `index.html` no navegador
3. O site funcionará localmente

### Instalação em Servidor
1. Envie os arquivos para o servidor web
2. Certifique-se que o servidor serve arquivos estáticos
3. Acesse o domínio configurado

### Otimização
- Comprima imagens para melhor performance
- Use CDN para Font Awesome
- Configure cache do servidor
- Minifique CSS e JS para produção
